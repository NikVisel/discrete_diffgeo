# Contributing to discrete_differential

Thank you for your interest in contributing to the discrete_differential project! We welcome contributions from the community and are grateful for your support.

## How to Contribute

### Reporting Issues

If you encounter any issues or bugs while using the crate, please report them by opening an issue on the [GitHub repository](https://github.com/NikVisel/discrete_diffgeo/issues). When reporting an issue, please provide as much detail as possible, including:

- A clear and descriptive title
- A detailed description of the issue
- Steps to reproduce the issue
- Any relevant error messages or logs
- The version of the crate you are using

### Requesting Features

If you have a feature request or an idea for an improvement, please open an issue on the [GitHub repository](https://github.com/NikVisel/discrete_diffgeo/issues) and provide a detailed description of the proposed feature or improvement. We appreciate your feedback and will consider all feature requests.

### Contributing Code

If you would like to contribute code to the project, please follow these steps:

1. Fork the repository on GitHub.
2. Clone your forked repository to your local machine.
3. Create a new branch for your contribution:
   ```bash
   git checkout -b my-contribution
   ```
4. Make your changes and commit them with clear and descriptive commit messages.
5. Push your changes to your forked repository:
   ```bash
   git push origin my-contribution
   ```
6. Open a pull request on the original repository and provide a detailed description of your changes.

### Development Environment Setup

To set up the development environment for the discrete_differential project, follow these steps:

1. Ensure you have [Rust](https://www.rust-lang.org/) installed on your machine. You can install Rust using [rustup](https://rustup.rs/).
2. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/NikVisel/discrete_diffgeo.git
   ```
3. Navigate to the project directory:
   ```bash
   cd discrete_diffgeo
   ```
4. Build the project:
   ```bash
   cargo build
   ```
5. Run the tests:
   ```bash
   cargo test
   ```

### Code Style

Please follow the Rust coding conventions and style guidelines when contributing code. We recommend using [rustfmt](https://github.com/rust-lang/rustfmt) to format your code and [clippy](https://github.com/rust-lang/rust-clippy) to lint your code.

### Code of Conduct

Please note that this project adheres to a [Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project, you agree to abide by its terms.

Thank you for contributing to the discrete_differential project!
